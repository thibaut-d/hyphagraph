# Bootstrap Command

Use this command at the beginning of an agent session.

## Required reads

1. `README.md`
2. `.ai/AI.md`
3. `.ai/ARCHITECTURE.md`
4. `.ai/CODE_GUIDE.md`
5. `TODO.md`
6. `.ai/TODO.md`
7. `.ai/AUDITS.md`

## Output

Summarize briefly:
- repository purpose
- architecture shape
- key coding rules
- live task status from `TODO.md`
- temporary file policy

Confirm that future work will follow:
- repository architecture
- typed, explicit boundaries
- `.temp/` for temporary artifacts
- `TODO.md` as the live handoff file

Do not start implementing during bootstrap unless explicitly asked.
